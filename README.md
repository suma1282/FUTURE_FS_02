# Mini CRM - Client Management System

A full-stack CRM (Customer Relationship Management) web application built to manage client leads efficiently.

This project helps users store, organize, update, and track client information using a clean dashboard interface with complete CRUD functionality.


## 🚀 Live Demo

🔗 Live Website: https://future-fs-02-pied-ten.vercel.app/

## 📌 Features

- 🔐 User Authentication
- 📊 Interactive Dashboard Overview
- ➕ Add New Client Leads
- 📋 View All Client Records
- 🔍 Search Leads by Name or Email
- 🏷️ Filter Leads by Status
- ✏️ Update Existing Lead Information
- 🗑️ Delete Client Leads
- 📈 Automatic Dashboard Statistics Update

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone -- https://github.com/suma1282/FUTURE_FS_02.git
```


### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```


### Backend Setup

```bash
cd backend
npm install
npm start
```


## 🔗 API Endpoints

| Method | Endpoint | Description |
|------|---------|-------------|
| GET | /api/leads | Get all client leads |
| POST | /api/leads | Create new lead |
| PUT | /api/leads/:id | Update lead details |
| DELETE | /api/leads/:id | Delete lead |
